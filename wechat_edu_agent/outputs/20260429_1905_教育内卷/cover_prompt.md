# Cover Prompt

cover_prompt_cn: 一位神情焦虑的中产家长（无具体面部特征），站在一个拥挤的考试日历前，日历上密密麻麻排满考试图标，背景是模糊的教室和少量学生，形成对比，色调冷静偏灰蓝，突出冲突感但不过度夸张，风格写实，适合公众号封面。
cover_prompt_en: A concerned middle-class parent (no specific facial features) standing before a crowded exam calendar filled with dense icons of tests and exams, with a blurred classroom and a few students in the background, creating contrast, cool blue-gray tones, realistic style, suitable for WeChat public account cover.
negative_prompt: 恐怖、低俗、夸张、真实名人、学校logo、机构logo、血腥、暴力、卡通、动漫风格、侵权风格
suggested_layout: 居中构图，家长位于左侧或中央，考试日历占据右侧或背景，文字叠加在画面下方或留白处，保持简洁清晰。
cover_text: 孩子少了，考试却更多了？
